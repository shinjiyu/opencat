=============================================
  OpenCat Portable — 使用说明
=============================================

1. 解压后运行安装：
   - Windows: 双击 install.bat
   - macOS/Linux: 终端运行 ./install.sh

2. 安装完成后：
   - 双击 open-chat.html（Windows）
   - 或打开 token.json 中的 chat_url 链接

3. 即可开始聊天！

注意：
- 首次安装需要网络连接（安装依赖 + 获取 Token）
- 安装完成后，聊天功能通过网页使用

遇到问题？
- 检查网络连接
- 确认 Node 版本正确（tools/node/）
- 查看 token.json 确认 Token 已获取
